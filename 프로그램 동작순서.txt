[ 1. 실행순서 ]

Config.json 파일을 메모장으로 열어서 필요한 부분을 수정한다
monitoring_scheduler.exe 에서 config 파일 위치를 지정한다.
"모니터링표 생성" 버튼 클릭

[ 2. 각 일자별 모니터링 담당자 배정로직 ]

가능한 사람 중
1순위: 해당 로테이션(평일/주말) 횟수 적은 사람
2순위: 전체 당직 횟수 적은 사람
3순위: 기존 순번상 앞 사람

[ 3. 모니터링 담당자 배정옵션]

1. 대체휴무일 모니터링 제외
2. 퐁당퐁당 X
3. 위 조건을 만족한 균등배정

[ 4. 대체휴무일 부여 우선순위 ]

1. 토요일 근무자
2. 일요일 근무자
3. 공휴일 근무자 순
